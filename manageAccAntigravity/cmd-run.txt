=== CLAUDE CODE CLI ===
# Mở PowerShell hoặc CMD và chạy:

set ANTHROPIC_API_KEY=sk-antigravity
set ANTHROPIC_BASE_URL=http://127.0.0.1:8045

# Hoặc 1 dòng (PowerShell):
$env:ANTHROPIC_API_KEY="sk-antigravity"; $env:ANTHROPIC_BASE_URL="http://127.0.0.1:8045"; claude


=== CODEX CLI ===
set OPENAI_API_KEY=sk-antigravity
set OPENAI_BASE_URL=http://127.0.0.1:8045/v1
codex


=== PYTHON OPENAI SDK ===
# Cài đặt:
pip install openai

# Code mẫu:
python -c "import openai; c=openai.OpenAI(api_key='sk-antigravity',base_url='http://127.0.0.1:8045/v1'); print(c.chat.completions.create(model='gemini-3-flash',messages=[{'role':'user','content':'Hello'}]).choices[0].message.content)"


=== CURL TEST ===
# Kiểm tra proxy hoạt động:
curl http://127.0.0.1:8045/v1/models

# Gửi request:
curl -X POST http://127.0.0.1:8045/v1/chat/completions -H "Content-Type: application/json" -H "Authorization: Bearer sk-antigravity" -d "{\"model\":\"gemini-3-flash\",\"messages\":[{\"role\":\"user\",\"content\":\"Hello\"}]}"

===============================================================
                   LƯU Ý QUAN TRỌNG
===============================================================
1. Phải mở Antigravity Tools và bật API Proxy trước khi chạy lệnh
2. Phải thêm ít nhất 1 tài khoản Google (OAuth) vào app
3. Nếu gặp lỗi 429:  đợi quota reset hoặc thêm nhiều account
4. Nếu gặp lỗi 404:  kiểm tra model name đúng chưa
